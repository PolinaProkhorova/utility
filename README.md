# Text Statistics Utility

Эта утилита на C подсчитывает количество строк, слов и символов в указанном текстовом файле. Она полезна для анализа текстового контента и может быть использована для различных целей, таких как оценка длины документа или подсчет количества слов.

## Возможности

- Подсчитывает количество строк в файле.
- Подсчитывает количество слов в файле.
- Подсчитывает количество символов в файле.
- Обрабатывает ошибки при неправильном использовании.

## Как использовать

1. **Скомпилируйте программу**:
   
   Убедитесь, что у вас установлен компилятор C (например, `gcc`). Используйте следующую команду для компиляции:
```
bash
gcc -o text_stats src/main.c src/stats.c
```

Эта команда создаст исполняемый файл с именем text_stats.

## Запустите программу:

Для запуска программы вам необходимо указать имя текстового файла в качестве аргумента. Используйте следующую команду:

```./text_stats <имя_файла>```
Например, если у вас есть файл example.txt, выполните:

```./text_stats example.txt```
## Ошибки:

Если вы не указали файл, программа сообщит об ошибке и подскажет, как правильно использовать.
Если файл не удается открыть, вы получите сообщение об ошибке с указанием причины (например, файл не найден).
## Пример использования
После выполнения программы для файла example.txt вы можете увидеть следующий вывод:

```
Добро пожаловать в утилиту подсчета статистики текста!  
Эта программа подсчитывает количество строк, слов и символов в указанном текстовом файле.

Статистика для файла 'example.txt':  
Строки: 5    
Слова: 20    
Символы: 100

Спасибо за использование утилиты!
```
